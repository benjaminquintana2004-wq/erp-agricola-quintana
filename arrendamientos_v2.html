API Key de Gemini · gratuita, sin tarjeta

Necesaria para leer contratos y facturas. Completamente gratuita con hasta 500 PDFs por día. Se guarda solo en tu navegador.

1
Entrá a aistudio.google.com/apikey con tu cuenta de Google
2
Hacé clic en Create API Key · No pide tarjeta de crédito
3
Copiá la clave (empieza con AIza) y pegala arriba

Instalación del backend de Google Sheets

1
Creá una planilla en Google Sheets con 3 hojas: Arrendadores, Movimientos y Contratos
2
En Sheets: Extensiones → Apps Script → pegá el contenido del archivo apps_script_backend.js
3
Implementar → Nueva implementación → Aplicación web → Cualquier usuario
4
Copiá la URL generada y pegala en el campo de arriba
